# TechNews365 OS Essentials Secure Edition

TechNews365 OS Essentials Secure Edition est une distribution Linux portable, légère et entièrement chiffrée, conçue pour fonctionner depuis une clé USB ou un SSD externe.  
Basée sur Linux Mint 22.3, elle offre un environnement nomade **sécurisé**, **persistant** et **rapide**, idéal pour les techniciens, administrateurs, étudiants et utilisateurs mobiles.

Cette édition met l’accent sur :
- la sécurité,
- la confidentialité,
- la portabilité,
- la simplicité d’utilisation.

---

## 🔐 Sécurité maximale

La Secure Edition intègre :

- **Chiffrement complet LUKS**  
  Vos données, paramètres, documents et configurations sont protégés par un chiffrement fort.

- **Persistance chiffrée**  
  Tout ce que vous faites est conservé de manière sécurisée entre les sessions.

- **Mode nomade sécurisé**  
  Utilisable sur n’importe quel PC sans laisser de traces.

- **Isolation des données**  
  Rien n’est stocké sur l’ordinateur hôte.

## 🛠️ Installation sur clé USB

1. Téléchargez l’ISO Secure Edition  
2. Flashez-la sur une clé USB (Ventoy, BalenaEtcher, Rufus…)  
3. Créez une partition **persistence**  
4. Activez le **chiffrement LUKS**  
5. Ajoutez le fichier `persistence.conf`  
6. Démarrez sur n’importe quel PC

Un guide détaillé sera disponible dans `/docs`.
