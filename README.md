# TechNews365 OS Essentials Portable

TechNews365 OS Essentials Portable est une distribution Linux légère, rapide et entièrement portable, basée sur Linux Mint 22.3.  
Elle est conçue pour fonctionner directement depuis une clé USB avec **persistance**, permettant de conserver vos fichiers, paramètres et applications entre chaque redémarrage.

Cette édition est idéale pour :
- les techniciens et administrateurs système,
- les étudiants,
- les utilisateurs nomades,
- les environnements de dépannage,
- les postes publics ou non sécurisés.

---

## 🚀 Fonctionnalités principales

- 🔹 **Basée sur Linux Mint 22.3** (Ubuntu 24.04 LTS)
- 🔹 **Version ultra‑légère** : rapide, stable, sans bloatware
- 🔹 **Mode portable** : fonctionne depuis une clé USB ou SSD externe
- 🔹 **Persistance intégrée** : vos données et paramètres sont conservés
- 🔹 **Compatible avec le chiffrement LUKS** (édition Secure)
- 🔹 **Environnement Cinnamon optimisé**
- 🔹 **Démarrage rapide et faible consommation RAM**
- 🔹 **Outils essentiels préinstallés**
- 🔹 **Branding TechNews365 OS**

## 🧰 Utilisation recommandée

1. Téléchargez l’ISO depuis SourceForge  
2. Flashez-la sur une clé USB (Ventoy, BalenaEtcher, Rufus…)  
3. Activez la **persistance**  
4. (Optionnel) Activez le **chiffrement LUKS**  
5. Démarrez sur n’importe quel PC avec votre environnement personnel
